NutroVia

Plataforma integral de Nutrición y Entrenamiento

Descripción del Proyecto

NutroVia nació con la idea de democratizar el acceso a planes de salud personalizados. A diferencia de otras aplicaciones genéricas, este sistema analiza parámetros físicos del usuario para calcular macros exactos y generar rutinas adaptadas al nivel de experiencia, disponibilidad de material y objetivos específicos (volumen, definición, mantenimiento).

Proceso Creativo y Desarrollo

El desarrollo comenzó definiendo la lógica del negocio en el backend mediante TypeScript y Express. Un reto importante fue el algoritmo de generación de dietas, que debía cuadrar alimentos de una base de datos nutricional con los requerimientos calóricos exactos de cada perfil, sin caer en la monotonía de recomendar siempre lo mismo.

Para el frontend, se buscó una interfaz sumamente limpia que no agobiara al usuario con un exceso de datos numéricos. Se implementaron dashboards intuitivos donde la progresión visual es la protagonista.

Identidad Visual y Colores (Pantone)

La paleta de colores de NutroVia está diseñada para inspirar vitalidad, salud y confianza, mezclando un verde "orgánico/tecnológico" con fondos muy oscuros para evitar la fatiga visual.
