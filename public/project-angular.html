Angular Task Manager

Gestión de agenda y autenticación avanzada

Descripción del Proyecto

Este showcase de Angular fue diseñado no solo como una herramienta funcional para la organización diaria web, sino también como un lienzo técnico para demostrar competencia en el ecosistema Angular y TypeScript. Permite crear flujos de trabajo eficientes controlando estados complejos y gestión de usuarios.

Proceso Creativo y Desarrollo

En este proyecto enfrenté el desafío de integrar una autenticación robusta manteniendo un diseño responsivo impecable. El proceso técnico implicó:

  • Manejo de estados a través de servicios inyectables (RxJS observables) garantizando tiempos de respuesta inmediatos.
  • Guards (canActivate) y JWT: Protección total de las rutas de usuario según su sesión.
  • Modularización: Arquitectura escalable y mantenible mediante la "Angular Way" utilizando Componentes y Módulos.

Identidad Visual y Colores (Pantone)

Dado que se trataba de una herramienta orientada a la productividad corporativa, se decidió prescindir de colores eléctricos excesivos (excepto para los botones de acción primarios) optando por azules profundos característicos de la marca de Google (Angular).
